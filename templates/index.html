New Light Ping Page

Welcome! This is the latest update to New Light! You can now remotely ping a server without having access to the Discord site. Simply input the link in the textbox, select the clan you wish to ping, and click Ping Server! You will be redirected right back here, so you can ping multiple servers if you wish.

This runs on a 30 Second Loop. Every 30 Seconds this program sends every link it has been given to the proper server with the @here ping.
